## Get started in three steps

Get your documentation site running locally and make your first customization.

### Step 1: Set up your Stream Deck Software

<Accordion title="Install the Stream Deck Software" description="(Skip this if you already have it installed)" icon="1" iconType="regular">
  <Tabs>
    <Tab title="Windows/Mac">
      ### You can download the Official Elgato Stream Deck Software for Windows and Mac by [clicking here](https://edge.elgato.com/egc/windows/sd/Stream_Deck_7.2.0.22454.msi). Remember to search your version

      Note: You will need Administrator Permissions. If you don't have, you will be asked for Administrator PIN/Password.
    </Tab>
    <Tab title="Other PC OS">
      I'm sorry, but there is no Official Software for other OS. Try in other computer or installing Windows/Mac
    </Tab>
  </Tabs>
</Accordion>
